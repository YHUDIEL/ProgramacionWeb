# Programación Web — Actividad 1

Repositorio con los ejercicios prácticos de la materia **Programación Web**, correspondientes a la Actividad 1 del Tema 2.

## Ejercicios incluidos

| # | Tema | Archivo |
|---|------|---------|
| 1 | Hola Mundo — Estructura HTML5 básica | ejercicio1.html |
| 2 | Párrafos y Formato — `<strong>`, `<em>`, `<br>` | ejercicio2.html |
| 3 | Enlaces Simples — `target="_blank"` y `_self` | ejercicio3.html |
| 4 | Imágenes Básicas — `src`, `alt`, `width` | ejercicio4.html |
| 5 | Blog Personal — Integración texto, imagen y enlace | ejercicio5.html |
| 6 | Secciones Semánticas — `<header>`, `<section>`, `<footer>` | ejercicio6.html |
| 7 | Listas Anidadas — `<ul>`, `<ol>`, `<li>` anidados | ejercicio7.html |
| 8 | Horario de Clases — Tabla HTML con CSS externo | ejercicio8.html |
| 9 | Formulario de Contacto — HTML5 avanzado + CSS | ejercicio9.html |
| 10 | Calculadora Celsius→Fahrenheit — HTML + JavaScript | ejercicio10.html |

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla)
